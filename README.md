# Student Registration Form

A simple React application built with Create React App that demonstrates a student registration form with client-side validation. This repository contains a small UI for collecting student details (name, email, course selection, and password) and shows how to validate user input, reset fields, and display feedback on submission.

## Features
- Student registration form (Full name, Email, Course dropdown, Password)
- Client-side field validation
- Form reset after successful registration
- Built with React and styled with CSS

## Tech Stack
- React (Create React App)
- JavaScript (ES6+)
- CSS

## Getting Started
These instructions will help you run the project locally.

### Prerequisites
- Node.js (>= 14.x recommended) and npm

### Install
1. Clone the repository locally:
```powershell
git clone https://github.com/hardikavijaykumarraut-lab/student-registration.git
cd "Student registration/student-registration"
```
2. Install dependencies:
```powershell
npm install
```

### Run locally
```powershell
npm start
```
Open http://localhost:3000 to view the app in the browser.

### Build
To build production files:
```powershell
npm run build
```

### Test
```powershell
npm test
```

## Project Structure
- `src/` — React source files
- `src/StudentRegistrationForm.js` — Main form component with validation
- `src/StudentForm.css` — Styles for the registration form
